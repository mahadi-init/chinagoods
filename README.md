# China Cup

An e-commerce admin panel

### Used technology
#### UI
- React
- NextJS
- Typescript
- TailwindCSS
- Shadcn
- React Hook Form
- Zod
- SWR
- JWT

#### Backend
- ExpressJS
- Mongodb
- JWT

### How to get started

Install npm dependencies </br>
` npm install `

Run the server </br>
` npm run dev `

Open the project in the browser </br>
`http://localhost:3000`

Build the project </br>
` npm run build `
